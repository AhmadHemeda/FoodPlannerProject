# FoodPlannerProject
<h2>Foodie</h2>
- Native Mobile Android Application to set weekly food plan, open source and free,using design architecture MVP, java rx, room , firestore.

<hr>

<h2>Application Features :</h2>
-  Onboarding Screen

-  Splash Screen

-  Login and Register

-  Daily Inspirations 

-  Meals planned for today 

-  Search by all Meals 

-  Search by areas

-  Search by ingrediants 

-  Search by category

-  User can save meals 

-  Add meals to week plan 

- User can LogOut 

- User can join as guest 

- May work locally 

<hr>
<h2>Permissions</h2>

- Network Permission.

<hr>
<h2>Tools </h2>

- Retrofit rx 

- TheMealDB Api

- room rx 

- firestore 

- mvp 

- trello
